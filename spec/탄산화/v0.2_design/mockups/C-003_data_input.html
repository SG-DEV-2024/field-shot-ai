입력 검증 (실시간)

  • 측정값 · 숫자(소수 허용) 1개. 빈 값이면 저장 비활성
  • placeholder · "0.00" (구체적 예시 값은 의도적으로 안 함 — 학습 다양성 유도)
  • 저장 버튼 활성 조건 · 값이 숫자로 파싱 가능. 음수도 허용 (현장 상황에 따라)
  • 비고 · 선택 입력. 저장 활성 조건에 영향 없음

저장 동작 — SurveyRecord

"저장 후 계속 촬영" 탭 시 다음 데이터 모델로 변환되어 StorageService.I.addRecord() 호출 후 Get.back()으로 C-001(카메라)로 복귀.

SurveyRecord.id'${DateTime.now().millisecondsSinceEpoch}'
SurveyRecord.surveyTypeSurveyType.carbonation
SurveyRecord.photoPathC-001에서 캡처된 jpg 경로
SurveyRecord.value측정값 (double)
SurveyRecord.note비고 textarea 값 (trim)
SurveyRecord.timestampDateTime.now()
SurveyRecord.uploadStatusUploadStatus.pending
SurveyRecord.measurementSubtypenull (탄산화는 subtype 없음)
SurveyRecord.annotationnull (탄산화는 좌표 annotation 없음)
