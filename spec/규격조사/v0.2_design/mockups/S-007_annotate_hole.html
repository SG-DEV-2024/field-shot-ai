4단계 진행 모델 (처음 찍기)

  • 진입 시 · 두 마커 모두 hidden 상태로 시작 · 단계 카드 모두 미완료(✓ 없음) · 다음 버튼 비활성
  • 1홀 입구 기준점 · 사진 첫 탭으로 홀 입구 amber 마커 표시 → 1단계 ✓ 활성 · 좌표 hole_lip_pixel
  • 2읽기 위치 · 사진 두 번째 탭으로 파랑 마커 표시 → 2단계 ✓ 활성. 줄자상 측정값(visible_reading_value)이 표시된 정확한 눈금 위치 · 좌표 tape_reading_pixel
  • 이후 사진 탭 · 두 마커 모두 표시된 상태에서는 가장 가까운 마커가 그 위치로 이동 (정밀 보정)
  • 3시작점 가시성 · 라디오 — visible (면에 노출) / hidden (가려짐). 줄자 0점 위치 추정에 사용
  • 4줄자 끝 접촉 여부 · 라디오 — yes (바닥 접촉) / no (떠 있음). 깊이값 신뢰도 메타데이터
  • 완료 시그널 · 카드 초록 톤(#bbf7d0 + #f0fdf4) + ✓. 1단계는 amber, 2단계는 blue 톤 유지 후 초록 ✓만
  • 다음 버튼 · 4단계 모두 완료 시에만 활성 (검증)
